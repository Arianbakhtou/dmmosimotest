
# Telegram DM Forward Bot
A Telegram Bot written in Pyrogram to Forward Messages Through Telegram Bot. 


## Features

- Forward Message to Your Chat
- Save Yourself From Thousand of DM's 

## Deploy to Heroku
- Please fork this repo and rename
- Then deploy from your repo


[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)
## How to Run This Project Locally

To run this project 
- Edit API_ID , API_HASH ,BOT_TOKEN, USERNAME from main.py 

- Then run these commands below
```bash
  pip install -r requirements.txt
```

```bash
  python main.py
```
